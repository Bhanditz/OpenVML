# OpenVML

OpenVML is an open soruce vector math library.

## Support Processors

* Generic platform
  * C and unoptimized codes

* x86_64
  * Intel Sandy Bridge
  * Intel Haswell

* ARM
  * ARMv7 (Cortex-A9, Cortex-A15)

## Compile

```
mkdir /your/build
cd /your/build
cmake /path/to/OpenVML
make
make install
```
